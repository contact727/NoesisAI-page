- Si la variable est définie, le formulaire envoie en `POST` JSON à cet endpoint avec ces clés:
  {
    "firstName": "...",
    "lastName": "...",
    "email": "...",
    "company": "...",
    "consent": true,
    "source": "website:noesisai",
    "path": window.location.pathname,
    "timestamp": new Date().toISOString()
  }
- Gérer les statuts (200/201 = success, sinon afficher une erreur).
- Documenter dans README comment créer l’endpoint Formspree + activer l’auto-réponse email.
