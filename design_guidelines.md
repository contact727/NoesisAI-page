# NOESIS AI Design Guidelines

## Design Approach
**Reference-Based Approach** - Drawing inspiration from modern B2B automation and AI service providers like Linear, Notion, and Vercel. Focus on clean, professional aesthetics that build trust while showcasing technical sophistication.

## Core Design Elements

### Color Palette
- **Primary**: 220 85% 12% (Deep navy blue)
- **Secondary**: 220 25% 25% (Muted slate)
- **Accent**: 200 100% 50% (Electric blue)
- **Background Light**: 0 0% 98% (Off-white)
- **Background Dark**: 220 25% 8% (Dark navy)
- **Text**: 220 15% 15% (Dark gray) / 0 0% 95% (Light gray for dark mode)

### Typography
- **Primary**: Inter (Google Fonts) - clean, modern sans-serif
- **Display**: Space Grotesk (Google Fonts) - for headlines and hero text
- **Hierarchy**: H1 (3xl-4xl), H2 (2xl-3xl), H3 (xl-2xl), Body (base-lg)

### Layout System
**Tailwind Spacing**: Primary units of 4, 8, 12, 16, 24 for consistent rhythm
- Container max-width: 1280px
- Section padding: py-16 lg:py-24
- Component spacing: gap-8 lg:gap-12

### Component Library
- **Navigation**: Fixed header with subtle blur backdrop
- **Buttons**: Primary (filled), Secondary (outline with blur on images)
- **Cards**: Subtle shadows with hover elevation
- **Forms**: Clean inputs with focus states
- **Data Display**: Clean tables and stat blocks

## Visual Treatment
**Professional Minimalism** with strategic use of:
- **Gradients**: Subtle blue gradients (220 85% 12% to 200 100% 50%) for hero backgrounds
- **Contrast**: High contrast for accessibility, dramatic section transitions
- **Background**: Subtle geometric patterns or grid overlays in hero sections

## Images
- **Hero Image**: Large, high-quality image showing automation/AI workflow visualization or modern office setup
- **Case Study Images**: Screenshots of automation dashboards, before/after workflow diagrams
- **Team Photos**: Professional headshots with consistent lighting
- **Service Icons**: Simple, outlined icons from Heroicons
- **Blog Images**: Relevant tech/automation stock photography

## Key Sections (Landing Page)
1. **Hero**: Full viewport with gradient overlay, compelling headline, CTA
2. **Value Proposition**: 3-column feature grid with icons
3. **Case Studies**: Showcase with metrics and testimonials
4. **Services Overview**: Brief intro with link to services page
5. **Contact CTA**: Simple, focused conversion section

**Constraint**: Maximum 5 sections, optimized for conversion without overwhelming users.
